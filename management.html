Deployment Manual
For Chatbot
  • Download the code
  • run “npm install”
  • run “node .”
  • The bot should be live on port 3000
For Flask
  • Download the code
  • additionally download a pretrained glove model
  • run “docker-compose up”
  • test endpoint at http://:5000/predict/
Connecting both:
  • Change the skills/newrequests….py to the flask link http://:5000/predict/
